# ZPluginHidder
ZPluginHidder is a Minecraft plugin designed to make server customization easier for users. With this plugin, users can fully customize their server settings and output messages, including customizing the /plugins command and the command not found message.

# Installation
To install ZPluginHidder:

Download the ZPluginHidder-1.0-SNAPSHOT.jar file.
Drop the jar file in the plugins folder on your Minecraft server.
Restart your server to activate the plugin.
Configuration
To customize the settings for CustomServer:

Open the ZPluginHidder folder.
Edit the config.yml file to customize your server settings.
Save your changes to the config.yml file.
Reload Configuration
To reload the configuration file:

Type the command /zpluginhidder or /zh in the Minecraft chat.
The configuration file will be reloaded and the new settings will take effect.
Notes
Please note that additional customization options may be available depending on the specific configuration options selected.
